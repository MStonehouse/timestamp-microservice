Timestamp Microservice

A Small Timestamp Api to Return Both Unix and UTC Time in a JSON Format

Enter a date in the input box below and click "Submit to API" to get your timestamp.





If you choose to use this manually, the endpoint for the timestamp is
https://solar-jodhpur.glitch.me/api/timestamp/

Just enter the endpoint, with the date you would like to use at the end of it, as a URL search and you will get a timestamp as a response. If the date is left empty and you just enter the endpoint it will return a time stamp with the current time. Examples follow.


To Convert from Unix Time.

Example: https://solar-jodhpur.glitch.me/api/timestamp/946684800000

Result: {unix: 946684800000, utc: "Sat, 01 Jan 2000 00:00:00 GMT"}


To Convert from UTC Time.

Example: https://solar-jodhpur.glitch.me/api/timestamp/01 Jan 2000

Result: {unix: 946684800000, utc: "Sat, 01 Jan 2000 00:00:00 GMT"}


An Empty Field Returns the Current Date.

Example: https://solar-jodhpur.glitch.me/api/timestamp/

Result: {unix: 1575849600000, utc: "Mon, 09 Dec 2019 00:00:00 GMT"}



Thanks and enjoy.
